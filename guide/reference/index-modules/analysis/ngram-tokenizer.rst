 NGram Tokenizer 
=================

A tokenizer of type ``nGram``.

The following are settings that can be set for a ``nGram`` tokenizer
type:

Setting
Description
``min_gram``
Defaults to ``1``.
``max_gram``
Defaults to ``2``.
